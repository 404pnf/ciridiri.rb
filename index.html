ciridiri.rb

ciridiri.rb is a Ruby+Sinatra port of ciridiri, dead simple wiki engine.

Requirements

  • sinatra

Installation

git clone git://github.com/vast/ciridiri.rb.git
cd ciridiri.rb
ruby ciridiri.rb

And point your browser to http://localhost:4567/.

Usage

Create new pages through accessing http://localhost:4567/path/to/new/page.html. Edit existent page through accessing http://localhost:4567/existent/page.html.e or just press ctrl-shift-e.
